# My First Website

This is my **first web development project**, created using **Semantic HTML** and **CSS**.  
The goal of this project is to practice building a simple, structured, and responsive web page using only HTML and CSS.

---

##Project Overview

This website demonstrates:
- Use of **semantic HTML elements** (like `<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>`).
- Basic **CSS styling**, including:
  - Colors, fonts, and layout
  - The **box model**
  - **Links and hover effects**
  - Proper **padding**, **margin**, and **borders**

---

## What I Learned

- Structuring web pages using **semantic HTML** tags.  
- Designing a clean and readable layout using **CSS**.  
- Understanding the **box model**, **selectors**, and **typography**.  
- Building a **navigation bar** without using JavaScript.  

---

##  Technologies Used

- **HTML5**  
- **CSS3**
